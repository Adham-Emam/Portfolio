List Comprehensions

List comprehensions provide a concise way to create lists. They are more readable and often faster than traditional for-loops. 

squares = [x**2 for x in range(10)];

Dictionary Comprehensions

Similar to list comprehensions, dictionary comprehensions offer a concise way to create dictionaries. 

squares = {num: num**2 for num in nums}

Generators and Generator Expressions

Generators allow you to iterate over data without storing the entire dataset in memory. This is useful for large datasets. 

def generate_numbers(n):
    for i in range(n):
        yield i

for num in generate_numbers(10):
    print(num)

Using enumerate

The enumerate function adds a counter to an iterable, which can be useful when you need both the index and the value in a loop. 

for i, fruit in enumerate(fruits):
    print(i, fruit)
